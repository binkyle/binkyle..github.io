## c++ 无符号右移运算
在统计一个数中的比特位为1的个数时，我们可以采用无符号右移即c>>>1。这是因为有符号右移时，低位截断，高位补符号位。而对于无符号右移，仅仅低位截断，这样我们就可以非常方便的获得其中1的个数。
